Welcome!

This website explains important Grade 7-9 Physics topics in detail: Kinematics, Thermal Energy, Static Electricity, and Electromagnetic Induction. It is designed to help students understand these topics clearly through easy-to-follow explanations, examples, and diagrams. Whether you are revising for class or exploring physics for the first time, this website provides all the information in one place.
